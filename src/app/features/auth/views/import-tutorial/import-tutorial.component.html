Trakt.TV

Per eseguire l'import dei dati da Trakt sara' sufficiente inserire il proprio nome utente.

Il nome utente puo' essere trovato seguendo i seguenti passaggi:

1.   Effettua il login

Effettua il login su Trakt

2.   Vai al tuo profilo

Dalla schermata a cui verrai reindirizzato, scegli "Your profile"

3.   Copia il tuo username dall'URL

Controlla nell'indirizzo della pagina. In fondo all'URL troverai il tuo nome utente.

Letterboxd

1.  Esegui il login

Esegui il login su Letterboxd

2.  Vai alla pagina delle impostazioni

Metti il mouse sul tuo nome in alto e seleziona "Settings" dal menu che si aprira'.

3.  Vai alla tab 'Data'

Tra tutte le opzioni disponibili nella pagina Settings, scegli "Data".

4.  Scarica i tuoi dati

Seleziona "Export your data".

Nella finestra di dialogo che si aprira', conferma la tua scelta cliccando nuovamente su "Export your data"

5.  Estrai la cartella compressa

Una volta completato il download, estrai la cartella compressa (.zip) che hai scaricato.

Puoi utilizzare programmi come WinRAR, WinZIP, 7Zip o le funzionalita' native di Windows.

6.  Carica il file watched.csv

Nella tua pagina profilo, carica il file watched.csv nell'area dedicata.
